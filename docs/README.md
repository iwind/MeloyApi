# MeloyAPI

MeloyAPI是使用`GO`语言编写的高性能API网关。

```
                 _________________
Client  <-->    | MeloyAPI Server |     <--> API Server
                |                 |
                | [ Proxy      ]  |
                | [ Statistics ]  |
                | [ Monitor    ]  |
                | [ Cache      ]  |
                 _________________ 
```
