# meloy-api help

打印帮助信息：

```bash
./meloy-api help
```

控制台上会输出类似以下的信息：

```
LiuXiangchaos-MacBook-Pro:main root# ./meloy-api help   
Usage:
  ./meloy-api
  	Start server sliently

  ./meloy-api debug
  	Start server in debug mode

  ./meloy-api start
  	Start server in system background

  ./meloy-api stop
  	Stop server

  ./meloy-api restart
  	Restart server

  ./meloy-api reload
  	Reload api configurations

  ./meloy-api version
  	Show api version

  ./meloy-api help
  	Show this help
```
